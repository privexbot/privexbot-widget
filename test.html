PrivexBot Widget Test

Use this page to test your deployed chatbot widget. Enter your chatbot ID and API key below.

How to Use

  1. Deploy a chatbot from the PrivexBot dashboard
  2. Copy the chatbot_id and api_key from the deployment response
  3. Paste them in the form above
  4. Click "Initialize Widget" to load the chat bubble
  5. Click on the chat bubble or "Open Widget" to start chatting

Embed Code for Your Website

Copy this code and paste it before the closing </body> tag:

<script src="YOUR_WIDGET_URL/widget.js"></script>
<script>
  pb('init', {
    id: 'YOUR_CHATBOT_ID',
    apiKey: 'YOUR_API_KEY',
    options: {
      baseURL: 'YOUR_API_URL',
      position: 'bottom-right',
      color: '#6366f1',
      greeting: 'Hello! How can I help you?',
      botName: 'Assistant'
    }
  });
</script>
